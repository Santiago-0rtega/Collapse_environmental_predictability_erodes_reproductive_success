Sensitivity analysis: excluding the 2015-2016 ENSO years

  • Show All Code
  • Hide All Code
  • View Source

Robustness of mismatch and nest-success models to years with very low breeding activity

This supplementary analysis evaluates whether the main phenological mismatch and nest-success inferences are sensitive to the unusually poor 2015 and 2016 breeding seasons. These years coincided with a strong ENSO event and had markedly reduced breeding activity, especially in 2016.

Motivation

Annual sample sizes were uneven across the study period, with 2015 and 2016 having unusually few breeding records relative to most years. To evaluate whether these extreme years disproportionately influenced the main results, we refit the retained model structures after excluding observations from 2015 and 2016.

The descriptive distribution of fledging outcomes by year is reproduced below as a visual reference for the imbalance in breeding activity and fledging outcomes across the time series.

Sensitivity model formulas

The sensitivity analyses use the same retained model structures as the main analysis, with the only data change being the exclusion of records from 2015 and 2016.

Phenological mismatch

The retained phenological mismatch model is the Gaussian location-scale m11 structure. It is refit separately for females and males:

Show mismatch sensitivity formula
mismatch_formula <- bf(
  X5MISMATCH ~ AGE + I(AGE^2) + TIME + AFR + LONGEVITY +
    (1 + AGE | q | RING) + (1 | YEAR),
  sigma ~ 1 + AGE + I(AGE^2) + TIME + (1 | q | RING)
)

Nest success

Nest success was modeled as the ordered fledging outcome (FLEDS: 0, 1, 2, or 3 fledglings) using the retained cumulative probit Q17 structure. It is also refit separately for females and males:

Show nest-success sensitivity formula
nest_success_formula <- bf(
  FLEDS ~ ZX5MISMATCH + I(ZX5MISMATCH^2) + TIME + AGE + I(AGE^2) + AFR + LONGEVITY +
    (1 | q | RING) + (1 | YEAR),
  disc ~ 1 + ZX5MISMATCH + AGE + TIME + (1 | q | RING)
)

Fitting script

The models are fit with the same sampling settings as the main analysis: 6,000 iterations, 2,000 warmup iterations, 4 chains, seed 123, zero initialization, adapt_delta = 0.99, and max_treedepth = 15. For speed, the supplementary script uses the cmdstanr backend with within-chain threading.

Show remote fitting command
# On totoro, from /home/ortegara/bfb_drop_enso_2015_2016:

  "Rscript",
  c(
    "Rscripts/sensitivity_drop_enso_2015_2016.R",
    "data/Lay_date_mismatches_250702.csv",
    "Rdata/drop_enso_2015_2016",
    "12"
  )
)
